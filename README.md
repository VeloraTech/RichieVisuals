# Creative Portfolio: RichieVisuals

A high-performance, modular portfolio section designed for a multidisciplinary creative. This layout is built to showcase high-end photography, cinematic videography, and professional branding in a single, balanced view.

# Features

- Fully Responsive: Uses a "stacking" logic that transitions from a 3-column desktop view to a single-column mobile view seamlessly.

- Container-Based: The entire section is wrapped in a single parent container, making it easy to drop into any existing website without breaking styles.

- Grid System: Implements a strict CSS Grid for the "Works" section to keep images perfectly aligned and square.

- Optimized for Media: Uses object-fit: cover to ensure images fill their spaces without stretching, regardless of original dimensions.

# Structure & "The Why"

The layout is divided into three strategic columns to give a 360-degree view of the creative's brand:

- Left (Vision): Uses a collage-style layout to show the "behind the scenes" or the artistic eye. It sets the tone for the creative's perspective.

- Center (Works): The focal point. It uses a 3x3 grid with a central logo slot to tie the brand identity directly to the physical output (e.g., product photography).

- Right (Mission): Features a wide, cinematic preview with a text overlay—ideal for showcasing video editing or storytelling capabilities.

# How to Use

- Placement: Copy the HTML/CSS into your project. The .rv-portfolio-component will automatically take up 100% width of whatever div you place it in.

- Image Collage: Best with 4:3 or 1:1 aspect ratio images.

- Grid: Use square images (1:1) for the best look.

- Cinematic: Use a 16:9 or wide-angle shot for the mission preview.

- Customization: Change the --background-color or font-family in the <style> tag to match the rest of your site.

# Tech Stack

- HTML5 (Semantic structure)
- CSS3 (Flexbox & CSS Grid)
- Google Fonts (System-standard clean sans-serif)
